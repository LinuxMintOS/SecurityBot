# SECURITY BOT

A Security Bot which helps you in Maintaining the Server.<br>Very Low chance of Server Nuked if the Bot is in the Server.

# Commands
### **Info Commands**
- s!botinfo
- s!userinfo
- s!serverinfo
- s!invite
- s!owner

### Moderation Commands
- s!lock
- s!unlock
- s!ban
- s!kick
- s!unban

### Security Number  Commands
- s!anti kick
- s!anti ban
- s!anti channelD
- s!anti channelC
- s!anti roleD
- s!anti roleC

### Security On/Off
- s!anti bot On
- s!anti bot Off

### Security Commands
- s!settings
- s!punishment
- s!punishment <kick|ban|remove role>
